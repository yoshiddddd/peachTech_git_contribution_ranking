## Overview  
サークルメンバーのgithub contribution数をランキング化したものです。　　

https://peach-tech-git-contribution-ranking-5krpgdb3y.vercel.app/
## Usage  
-React v18  
-node v18.19  
-github graphQL API  
-fire store  
-vercel  

## Description
github graphQL APIを使用して、各ユーザーのcontribution数を取得し、多い順にsortしたものを表示しています。  
サークル内のコミット活性化が目的です。
